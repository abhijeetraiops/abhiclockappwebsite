# abhiclockappwebsite
A Full Featured Clock Application, with Traditional and Ancient System of Time at its Core.

## Website files

- `index.html` - Landing page (features, screenshots, FAQ)
- `privacy.html` - Privacy policy page
- `styles.css` - Shared app-like dark/amber theme
- `app.js` - Small UI script (dynamic footer year)
- `assets/images/screenshots/` - App screenshots
- `assets/images/branding/` - Logo/app icon assets
- `assets/images/store/` - Optional listing graphics

## Deploy on GitHub Pages

1. Open repository **Settings** -> **Pages**
2. Under **Build and deployment**, choose:
   - **Source:** Deploy from a branch
   - **Branch:** `main`
   - **Folder:** `/ (root)`
3. Save, then wait for deployment
4. Use generated site URL in Play Console (privacy policy link can point to `/privacy.html`)

## Before publishing

- Replace `YOUR_SUPPORT_EMAIL` in:
  - `index.html`
  - `privacy.html`
